PROJET FINAL S3 - APPLICATION BNGRC

TO-DO LIST REPARTIE POUR 3 PERSONNES

1) ANALYSE DU PROJET

Application de suivi des collectes et dispatch de dons pour les
sinistrés. Objectif principal : Dashboard affichant : - Liste des
villes - Leurs besoins - Les dons attribués - Simulation de dispatch
automatique par ordre de date

Règles importantes : - Besoin = prix unitaire fixe + quantité - Pas
d’identification individuelle des sinistrés - Dispatch par ordre de
saisie (FIFO)

2.  Tables à créer

Table ville - id - nom

Table besoin - id - ville_id (FK) - type (nature / materiaux / argent) -
libelle (riz, huile, tôle…) - prix_unitaire - quantite - date_saisie

Table don - id - type - libelle - montant (si argent) - quantite (si
nature/materiaux) - date_saisie

Table dispatch - id - don_id - besoin_id - quantite_attribuee -
date_dispatch
